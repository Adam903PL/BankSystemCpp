# 🏦 BankSystemCpp

**BankSystemCpp** is a simple console-based banking simulation written in **C++**.

The application allows users to create and manage bank accounts, perform basic banking operations, and generate transaction reports. It was built as a practical project to explore object-oriented programming, file handling, and console application design in C++.

---

## 💡 Overview

The project simulates a basic banking system in the terminal, where users can interact with accounts through a menu-driven interface.

It supports core banking operations such as:

* creating accounts
* deposits and withdrawals
* transfers between accounts
* checking balances
* viewing transaction history
* saving and loading data from files
* generating reports

---

## 🔑 Features

### 👤 Account Management

* Create new bank accounts
* Store account owner details
* Keep track of account balances

### 💸 Banking Operations

* Deposit funds
* Withdraw funds
* Transfer money between accounts

### 📜 Transaction History

* View full transaction history for a selected account
* Sort and display past operations clearly

### 💾 File Support

* Save account data to files
* Load saved data back into the system

### 📊 Reporting

* Generate monthly reports
* Generate reports for a selected date range

### 🖥️ Console Interface

* Interactive menu-based navigation
* Clear text UI with custom banners and status messages
